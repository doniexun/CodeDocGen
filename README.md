# CodeDocGen
AutoGen CodeDoc with the given RootDir、ExceptDirs and fileExts.

# Dependencies
baliance.com/gooxml

# Usage
1. Install gooxml
go get baliance.com/gooxml/
go build -i baliance.com/gooxml/...

2. $ cd CodeDocGen/
   $ vi main.go
Specify the RootDir、ExceptDirs and fileExts in main.go

3. $ go run main.go
